Smart Parking System : This has three main modules i.e the Hardware ( Raspberry pi) , Database and the Software (Website). The website is developed using the softwares like Tomcat server, SQLyog, MySQL server and the Eclipse IDE. Programming languages used are Java, JSP and CSS. The website allows the user to book a vacant spot and once it's booked, a SMS is sent with a map to the booked parking spot. The hardware section consists of the Raspberry pi, the PIR sensors, LED lights (Red-parked, Green- Vacant, Yellow- booked) and RFID reader and the tag. Python language is used to code the hardware section.
